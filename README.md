# goalSetter app
MERN application with CRUD functionality and Authentication and Authorisation using JWT.

# Usage
Rename the .envexample to .env and add your MONGO_URI

# Install dependencies

*Backend deps*
`npm install`

*Frontend deps*
`cd frontend`
`npm install `

*Run server*
`npm run server`
